No [[CyberSecurity/Notas/DFIR/Windows/Windows|Windows]] podemos coletar dados volateis e não volateis.
Dados volateis (dados em RAM, por exemplo) podem ser recuperados quando o sistema está rodando ja [[CyberSecurity/fatec/Pesquisa/Pesquisa/ferramentas/os/os|os]] dados não volateis (dados no Hard Drive, por exemplo) podem ser recuperads mesmo depois que o sistema é destruido por uma falha de energia.
Varios [[softwares]] e tecnicas podem ser aplicados na coleta de dados como incluir dados volateis, revisitar o sistema de [[CyberSecurity/Course/forense/logs/logs|logs]] ou monitorar as [[CyberSecurity/fatec/2026/1-2026/Desenvolvimento/ATIVIDADES|ATIVIDADES]] do usuario. O objetivo é manter a acuracia e integridade das evidências coletadas.
Para manter a etica e integridade dos dados é importante que a coleta seja direcionada por um framework etico e legal.
