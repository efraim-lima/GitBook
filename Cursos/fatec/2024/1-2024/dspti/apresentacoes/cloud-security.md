# Cloud Security

O processo de [[Segurança]] de cloud é algo que deixa seus usuários mais tranquilos pois depende do provedor, mas também o usuário precisa ter cautela em como ele utiliza seus serviços, pensando em critérios de autenticação/acessos/disponibilidade.

A defesa de nuvem é feita em camadas, o usuário precisa segurar seus acessos e permissões, [[CyberSecurity/fatec/Pesquisa/Pesquisa/ferramentas/os/os|os]] servidores dos provedores possuem processo de [[Segurança]] de seu datacenter.

algumas [[CyberSecurity/fatec/Pesquisa/Pesquisa/ferramentas/ferramentas|ferramentas]] proporcionam [[Segurança]] para servidores cloud com focos diferentes para que o usuário possa ter capilaridade em sua defesa.
