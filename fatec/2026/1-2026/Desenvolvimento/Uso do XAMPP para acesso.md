# Processo de Acesso ao XAMPP 
Para acessar o XAMPP no linux devemos primeiro baixar o utilitário no website oficial https://www.apachefriends.org/download.html

Nisso basta seguir os demais [[comandos]] para que possamos instalar o fulano

`chmod 755 xampp-linux-*-installer.run`
`sudo ./xampp-linux-*-installer.run`


Tendo instalado este carinha podemos seguir para os próximos passos, para acesso ao ambeinte GUI

`xhost -si:localuser:root`
`sudo /opt/lampp/lampp restart`

Assim podemos acessar nossa interface grafica via

`sudo Downloads/xampp-linux-*-installer.run`
